NO PROFIL LULUSAN DESKRIPSI PROFIL
1 Senior Programmer Memimpin produksi pada suatu proyek pembuatan software sekaligus memastikan standar kualitas software yang dibuat dengan cara mengelola, membuat jadwal dan membagi setiap tugas kepada programmer sesuai dengan job diskripsi yang sudah ditentukan sebelumnya
2 Software Developer Membangun suatu software sesuai dengan konsep Software Development Life Cycle (SDLC)sesuai dengan kebutuhan klien untuk kebutuhan software komputer atau perangkat lain
3 Web Developer Merancang dan membuat suatu website sesuai dengan standar pembuatan website sekaligus bertanggung jawab dalam perawatan dan pemeliharaan.
4 Game Developer Merancang, membuat dan melakukan pemeliharaan suatu software game yang interaktif dengan menggabungkan teks, grafik, animasi, pencitraan, audio-video display, dan media lainnya
5 Database Administrator (DBA) Melakukan penyimpanan dan mengorganisasikan data dengan menggunakan perangkat lunak khusus serta memastikan bahwa data yang tersedia aman dan terlindungi dari akses ilegal.
6 System Analyst Melakukan analisis sistemdalam tahap pembuatan software beserta prosedur pembuatanuntuk memberikan solusisesuai dengan kebutuhan organisasi bisnis atau perusahaan
7 Software Tester Melakukan pengujian software menggunakan metode-metode sesuai dengan standar evaluasi dalam pembuatan software
